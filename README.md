# My Orders Project (Simple POS Web App)

A simple **Point of Sale (POS) web application** built to replace manual sales recording in a notebook.  
The app helps manage orders, customers, payments, and outstanding balances in an easy and user-friendly way.

---

## Features

- Products Management (add, delete, edit)
- Automatic order calculation
